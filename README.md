📊 Techno Sales Dashboard (Power BI Project)

📌 Project Overview

This project is an interactive Sales Analytics Dashboard built using Power BI.
The dashboard analyzes sales performance from 2020–2022 using a CSV dataset containing 5,000+ records.

The goal of this project is to derive business insights such as:

*Total Sales & Profit.      
*Top Performing Products.    
*Brand-wise Sales Distribution.    
*State-wise Sales Analysis.    
*Supervisor Performance Tracking.  



🗂 Dataset Information

File Used: Sales_Data.csv

Total Records: 5,000+

•Columns Included:

•Order Number

•State Code

•Customer Name

•Order Date

•Status

•Product

•Category

•Brand

•Cost

•Sales

•Quantity

•Total Cost

•Total Sales

•Assigned Supervisor



📈 Dashboard Features

✔ KPI Cards (Total Sales, Profit, Quantity, Transactions)

✔ Top 5 Products by Sales

✔ Quantity by Category (Bar Chart)

✔ Brand-wise Sales Distribution (Pie Chart)

✔ State-wise Sales Map (Geo Visualization)

✔ Supervisor Image Slicer (Dynamic Filtering)

✔ Interactive Filtering & Cross Highlighting
